## 8. AI Model Explanation  ⭐ (heavily marked — explain the logic clearly)

### Feature 1 — Smart Restock Prediction (`ai/restock.py`)
- For each product, build a daily-demand series over the analysis window
  (missing days = 0 demand).
- Estimate projected daily demand = average of (historical mean) and a
  **scikit-learn LinearRegression** trend projection.
- `daysToStockout = currentStock / projectedDailyDemand`.
- `recommendedRestock = projectedDailyDemand × (leadTime + safetyStock) − currentStock`.
- Output example: *"Milk may run out within 3 days. Recommended restock: 50 units."*

### Feature 2 — Fast & Slow Moving Analysis (`ai/movement.py`)
- Compute sales velocity (units/day) and revenue per product.
- **K-Means (k=2)** on standardised [velocity, revenue] → FAST vs SLOW.
- Products with no sales within the dead-stock window → DEAD STOCK.
- Output example: *"Product X has not sold for 45 days."* Visualised as a doughnut chart.

### Feature 3 — Sales Trend Analysis (`ai/trend.py`)
- Aggregate revenue by day / week / month (pandas resample).
- **LinearRegression** on daily revenue → 7-day forecast.
- Detect weekend lift and month-over-month change.
- Output example: *"Sales increase during weekends (+40%)."*

### Feature 4 — Intelligent Alerts (`ai/alerts.py`)
- Rule-based engine: LOW_STOCK, OVERSTOCK, EXPIRING (≤14 days),
  SALES_DROP (last 7 days < 50% of prior 7 days).
- Output example: *"Increase beverage stock before weekends."*
